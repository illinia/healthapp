Login, Singup, Calculating, Profile Functions
<br>
<br>

## Installation
---
*first terminal*
```
git clone https://github.com/illinia/healthapp.git
cd healthapp && npm i
cd ios && pod install && cd ..
npm start
```
> after first command finish, second terminal
```
cd healthapp && npm run ios
```
